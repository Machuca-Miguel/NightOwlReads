NightOwlReads
=============

NightOwlReads is a web application designed for book enthusiasts who wish to organize their personal libraries, connect with fellow readers, share reviews, and explore new books.

Features
--------

- **User Registration:** Users can create an account to access all features of the application.
- **Library Management:** Users can create and organize their personal libraries with their favorite books.
- **Social Connectivity:** Users can make friends, join book clubs, and share reviews and book recommendations.
- **Book Exploration:** Users can search for books by title, author, or genre, and explore detailed information about each book.
- **Achievement System:** Users can unlock achievements by completing certain actions, such as reading a specific number of books or writing reviews.
- **Statistics Dashboard:** Users can view statistics about their reading activity, including books read, time spent reading, and more.

Technologies Used
-----------------

- **Frontend:** Angular
- **Backend:** Node.js
- **Database:** MySQL
- **External API:** Goodreads API

Installation
------------

1. Clone this repository to your local machine.
2. Install frontend dependencies by running `npm install` in the `/frontend` directory.
3. Install backend dependencies by running `npm install` in the `/backend` directory.
4. Create a `.env` file in the `/backend` directory and configure necessary environment variables (e.g., Goodreads API key).
5. Start the backend server by running `npm start` in the `/backend` directory.
6. Start the frontend application by running `ng serve` in the `/frontend` directory.
